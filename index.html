Learning to create on the Web used to be easier. If you saw a great website, you could view the source, get the code and see how it was made. Many developers learned that way.

BUILD FOR THE WEB OF TOMORROW

It takes increasingly diverse skills to build the Web as the platform for innovation, opportunity and participation. It’s that breadth and depth of knowledge that we aim to share with those working on the leading edge

These days it’s a lot harder, as libraries and development practices are often hidden from the browser, but developers are as committed to learning, sharing and teaching each other as ever.

Let’s come together to view the source of the modern Web, build the Web we want and explore the best ways to apply new Web technologies. We hope to see you in Portland in November.
